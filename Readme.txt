1- Install python3 and pip3 with:
sudo apt-get python3
sudo apt install python3-pip

2- Install pandas for python with:
pip3 install pandas


3- Copy paste the content of this folder to your source code folder and run autograder.py with python3 in that directory: 
python3 autograder.py 


4- You should get at least few matches. If you get few false, it is fine. Current input files have few repetitive value. This will not happen in the grading benchmarks.

5- Running the autograder should take less than 5 minutes. (it should be around 30 seconds)

6- Number of entries in the input files might vary the grading benchmarks.
